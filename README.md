# FlorysDiaries

FlorysDiaries ist eine lokal orientierte Reise-App für Reiseplanung, Dokumente, Checklisten, Karten, Statistiken, Momente und Sicherungen.

## Aktueller Projektstand

Die Projektmetadaten sind auf **`v1.0.0+7`** synchronisiert.

- Release-App: `com.florysdiaries.app`
- Entwicklungs-App: `com.florysdiaries.app.debug`
- Aktive Entwicklung und Tests erfolgen in **FlorysDiaries DEV**.

## Entwicklung

```powershell
flutter pub get
flutter analyze
flutter test
flutter run
```
